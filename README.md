
# project setup




## Run Locally

Clone the project

```bash
  git clone https://github.com/kkapil94/chat-app.git
```

Go to the project directory

```bash
  cd chatApp
```

Install dependencies for client

```bash
  cd client
  npm install
```

Run the client side

```bash
  npm run dev
```

Install dependencies for server

```bash
  cd server
  npm install
```

Start the server 

```bash
  npm run dev
```
